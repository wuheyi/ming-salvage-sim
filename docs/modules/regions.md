# 地区模块

地区模块记录每个地区的基本盘。地区不用拆太细，先以京师、省、军镇和特殊交通线为主。

## 核心数值

- `民心`：当地百姓对朝廷的信任。
- `动乱`：盗匪、逃户、流民、抗税和叛乱风险。
- `税收`：本月可向中央提供的钱粮。
- `田地`：登记田地和隐田的大致规模。
- `人群`：该地区主要人群数量和状态。

## 地区关系

- 相邻：战乱、流民、疫病、盗匪会扩散。
- 漕运：影响粮食和税银北上。
- 商路：影响银流、物价和商税。
- 军事支援：一地失守会推高周边压力。

## 玩法重点

玩家不会直接调每个地区的小数值，而是从地方奏报、巡抚回奏、商人消息和锦衣卫密报里判断真实情况。
