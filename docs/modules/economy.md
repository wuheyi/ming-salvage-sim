# 经济模块

经济模块只追踪皇帝真正关心的账：国库有没有钱，内库能不能救急，本月收了多少，花了多少，欠了多少。

国库、内库不是 0-100 状态条，而是实际钱粮整数，MVP 暂用“万两”为单位。民心、皇威、边防等才是 0-100 局势量表。

## 核心数值

- `国库`：朝廷公开财政，主要用于军饷、赈灾、官俸、工程，单位为万两。
- `内库`：皇帝私人可调度的钱物，适合救急和密支，单位为万两。
- `月度收入`：本月实际入账。
- `月度支出`：本月实际花出。
- `欠账`：军饷、工程、赈灾或官俸拖欠。

每回合开始展示上一月总结：

- 国库、内库的月度收入、支出、净变化。
- 上一月发生的奏报事件。
- 已批准圣旨的执行结果。
- 核心数值变化。
- 对局势造成的影响。

## 简化账目

每笔账只记五项：

- 来源或用途。
- 金额，单位为万两。
- 经办人或经办衙门。
- 关联圣旨。
- 是否有截留、虚报或拖欠。

## 玩法重点

玩家不需要管理细碎账册，只需要通过大臣回奏判断：

- 钱从哪里来。
- 钱花到哪里去。
- 谁说钱不够。
- 谁可能截留。
- 要不要用内库补国库。
