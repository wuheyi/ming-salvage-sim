# 建筑模块

建筑模块记录地区内重要设施。建筑只保留少数核心数值，其他细节交给事件、回奏和 LLM 描述。

## 核心数值

- `等级`：建筑规模和能力，1-5。
- `完好`：能否正常运转，0-100。
- `产出`：每月带来的钱粮、军力、科技或民心效果。
- `维护`：每月需要的钱粮、人力或物资。
- `风险`：贪腐、事故、扰民、被毁或被占用的可能。

## 建筑类型

- 财政：税课司、钞关、盐仓、户部库。
- 军事：城墙、边堡、烽燧、军营、火器局、炮台。
- 民生：常平仓、义仓、水渠、堤坝、医馆、粥厂。
- 科技：工坊、铸炮厂、书院、译局、观象台、军校。
- 交通：驿站、码头、船坞、桥梁、漕运仓。
- 内廷：内库、织造、御用工坊、密档房。

## 玩法重点

皇帝只能通过圣旨命人修、建、查、拨款、停办或追责。建筑不是玩家直接摆放的棋子，而是地方和官员执行圣旨后的长期结果。
