# 科技模块

科技模块表示皇帝用未来记忆推动超时代想法。科技不是按钮，而是皇帝在召对中提出方向，再指定人物试办。

## 核心数值

- `皇帝认知点`：玩家每月可投入的科技理解和注意力。
- `超时代等级`：越高越难，越容易失败或被攻击。
- `项目进度`：从构想到试点、推广、成熟。
- `资源需求`：需要银、物资、工匠、场地或人物支持。
- `反弹风险`：被文官、士绅、工匠、军队或保守派抵制的风险。

## 科技流程

1. 皇帝提出想法。
2. 召见合适人物讨论可行性。
3. 下旨立项，指定负责人和资源。
4. 每月回奏进度。
5. 成功则进入试点，失败则损失钱粮、皇威或人物信任。

## 超时代原则

能超时代，但越超越越难。火器定装、账册审计、农作物推广这类容易落地；蒸汽机、近代银行、现代军工这类会因为材料、工匠、制度和理解不足而极难推进。
